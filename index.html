Tema 1: Completa la siguiente tabla

Concepto Descripción Concepto Descripción
Tabla en HTML Es una estructura que permite organizar información en filas y columnas dentro de una página web. Formulario en HTML Es un conjunto de campos que permiten la entrada de datos por parte del usuario.
Etiqueta <table> Define el inicio de una tabla en HTML. Etiqueta <form> Define un formulario para recolectar la información del usuario y enviarla a un servidor.
Atributos de <table> Atributos como border, cellpadding, cellspacing, entre otros, que modifican la apariencia de la tabla. Etiqueta <input> Define un campo para la entrada de datos, como texto, contraseña, checkbox, etc.
Etiqueta <tr> Define una fila dentro de una tabla. Etiqueta <label> Asocia una etiqueta de texto a un control de formulario, facilitando la accesibilidad.
Atributos de <tr> Atributos como align o valign que alinean el contenido de la fila. Etiqueta <select> Define una lista desplegable de opciones dentro de un formulario.
Etiqueta <td> y <th> <td> define una celda en una tabla, mientras que <th> define una celda de encabezado. Etiqueta <textarea> Define un área de texto para la entrada de múltiples líneas en un formulario.
Atributos de <td> y <th> Atributos como colspan, rowspan, align, entre otros, para personalizar celdas y encabezados. Etiqueta <fieldset> y <legend> <fieldset> agrupa controles de formulario y <legend> proporciona una descripción para el grupo.
Tablas avanzadas Incluyen elementos como el uso de colspan, rowspan, thead, tbody, y tfoot para mayor control. Etiqueta <meter> Representa un valor escalar dentro de un rango conocido, como una barra de progreso.
Etiqueta <thead> Define el encabezado de una tabla, que generalmente contiene etiquetas <th>. Etiqueta <progress> Muestra el progreso de una tarea en curso, como la descarga de un archivo.
Etiqueta <tfoot> Define el pie de tabla, que generalmente contiene datos finales o de resumen. Etiqueta <datalist> Proporciona una lista de opciones predefinidas para un campo <input>.
Etiqueta <tbody> Agrupa el contenido principal de la tabla. Etiqueta <summary> Proporciona un resumen visible cuando se utiliza dentro de un elemento <details>.

Tema 2: Elementos de la estructuración de una página web

  • Contenedor (wrapper): Agrupa y estructura visualmente el contenido de una página web.
  • Cabecera (header): Sección superior que incluye elementos como el logotipo y el menú.
  • Contenido (content): Parte central que muestra la información principal.
  • Menú (menu): Lista de enlaces que permiten navegar entre las secciones de la página.
  • Pie (footer): Parte inferior que contiene información secundaria o legal.
  • Lateral (sidebar): Barra lateral que muestra información adicional.
  • Etiqueta <div>: Contenedor genérico utilizado para estructurar contenido y aplicar estilos.

Tema 3: Estructuración semántica de una página web

  • <header>: Representa la cabecera de una sección o documento.
  • <nav>: Define una sección de navegación con enlaces importantes.
  • <section>: Agrupa contenido relacionado en una sección temática.
  • <article>: Contenido independiente que puede distribuirse de manera autónoma.
  • <aside>: Contenido relacionado pero independiente del contenido principal.
  • <footer>: Define el pie de una sección o documento, generalmente con información legal o enlaces secundarios.
  • <main>: Representa el contenido principal de la página.
  • <h1> - <h6>: Títulos y subtítulos jerárquicos para estructurar el contenido.
  • <figure> y <figcaption>: Agrupa y describe contenido visual como imágenes o gráficos.
  • <mark>: Resalta texto relevante.
  • <time>: Representa una fecha u hora específica.
  • <address>: Proporciona información de contacto o dirección.
